# Fatec - Aprendizagem de Máquina

Repositório desenvolvido durante a disciplina de **Aprendizagem de Máquina** da Fatec, contendo os projetos, exercícios e atividades práticas realizados ao longo das aulas utilizando **Python**, **Jupyter Notebook** e bibliotecas voltadas para Ciência de Dados e Machine Learning.

## Objetivo da Disciplina

A disciplina teve como objetivo apresentar os principais conceitos de aprendizado de máquina, análise de dados e inteligência artificial, utilizando ferramentas práticas para criação de modelos preditivos, classificação e análise estatística.

Durante as aulas foram desenvolvidas atividades utilizando bibliotecas como:

- NumPy
- Pandas
- Matplotlib
- Counter
- CSV
- Torch

Além de técnicas como:

- Regressão Linear
- Regressão Logística
- KNN (K-Nearest Neighbors)
- Árvore de Decisão
- Perceptron
- Multi Layer Perceptron (MLP)

# Como Executar o Projeto

## 1. Clonar o repositório

```bash
git clone https://github.com/AlissaGabriel/Fatec-AprendizagemDeMaquina.git
```

## 2. Abrir o projeto no Visual Studio Code

Abra a pasta do repositório no Visual Studio Code.

## 3. Instalar as dependências

```bash
%pip install notebook numpy pandas matplotlib torch
```

## 4. Executar o Jupyter Notebook

1. Escolha o arquivo .ipynb referente à aula ou atividade desejada;

2. Abra o notebook no Visual Studio Code;

3. No menu superior do notebook, clique em:
```bash
Run All
```
Assim todas as células do notebook serão executadas em sequência, reproduzindo os resultados do projeto.
